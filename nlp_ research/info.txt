오픈 AI의 API
    - 오픈 AI는 자신이 보유한 막대한 컴퓨터 자원으로 언어 모델인 GPT , 이미지 생성 모델인 달리 , 동영상 생성 모델인 소라 등의 대규모 모델을 실행하고
      사용자가 결과를 받아갈 수 있는 API 서비스를 제공한다.
    - 오픈 AI의 API를 이용하면 최신 모델을 누구나 이용할 수 있다.

오픈 API 라이브러리 설치
    - pip install openai==1.58.1

[.env]
    - pip install python-dotenv
    - 위와 같이 라이브러리를 설치하면 .env 파일에 있는 키-값 쌍을 읽어서 환경 변수로 설정해준다.

[프롬프트 엔지니어링]
    - 프롬프트 엔지니어링이란 GPT와 같은 언어 모델이 사용자가 원하는 답변을 하도록 유도하기 위해 문장을 입력하고 효과적으로 설계하는 작업을 의미한다.
    - GPT에게 같은 질문을 하더라도 사전에 어떤 프롬프트를 입력해 두었는지에 따라 답변이 완전히 달라질 수 있다.
    - API 요청시 role : system 의 content에 역할을 부여하면 그 역할에 맞춰 response를 준다.
    - GPT가 원하는 패턴에 맞춰 답변하도록 예시를 한 번 제시해서 유도하는 방식이 원샷 프롬프팅이다.
    - 퓨샷 프롬프팅 : 여러가지의 예를 전달허요 답변 유도

[멀티 턴]
    - GPT API는 message에 있는 내용을 기반으로 다음 응답을 생성하므로 message의 값이 바뀌지 않으면 gpt는 이전에 만든 답변을 기억하지 못한다.
    - 여러 번 대화(턴)할 때 이전 대화를 기억하고 적절하게 반응하는 것을 멀티턴이라고 한다.
    - 멀티턴은 언어 모델이 대화의 맥락을 유지하면서 사용자와 여러 차례 상호 작용할 수 있도록 하는 대화 방식이다.
